# TAFímetro

Este projeto é uma calculadora que utiliza funções para calcular notas e tempos baseados em dados de desempenho em corridas. O design é moderno e otimizado para dispositivos móveis, proporcionando uma experiência de usuário agradável.

## Estrutura do Projeto

- **index.html**: Documento HTML principal que estrutura a página da web.
- **css/styles.css**: Arquivo de estilos que define a aparência visual da página.
- **js/calc.js**: Contém as funções JavaScript para calcular notas e tempos.
- **js/app.js**: Script principal que interage com os elementos HTML e manipula a lógica da aplicação.

## Como Usar

1. Abra o arquivo `index.html` em um navegador web.
2. Insira os dados necessários (tempo, idade, sexo, distância) nos campos apropriados.
3. Clique no botão para calcular a nota ou o tempo.
4. Os resultados serão exibidos na página.

## Tecnologias Utilizadas

- JavaScript
